Software & ML Projects

Parkinson's Progression AI

Tools : Hugging Face, Docker, Streamlit

Parkinson's Progression AI is a tool to track PD Progressioon through sit-to-stand transitions, audio, and handwriting analyses. Analyzing real time data, the application gives power back to patients by tracking their progression in a quantitative way via a PD progression score. The three models ensembled to create the progression score focus on segmenting video to locate muscle atrophy, detecting voice variance, and identifying micrographia and rigidity in a handwriting sample.

Deployed application can be found here, and the project code can be found here.
